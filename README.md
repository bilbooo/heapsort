# Heapsort Visualizer

Visualizzazione interattiva dell'algoritmo **Heapsort** realizzata in HTML, CSS e JavaScript vanilla.

## Demo

Disponibile tramite GitHub Pages: `https://<username>.github.io/heapsort`

## Come funziona

Il progetto anima passo-passo la costruzione dello heap (max-heap) e la fase di estrazione, mostrando ogni swap effettuato sull'array.

## Utilizzo

Apri `index.html` nel browser oppure visita la GitHub Page del progetto.

## Stack

`HTML` · `CSS` · `JavaScript`
